## 搬瓦工会自动扣费吗？先搞清楚这个

很多人来搜"搬瓦工怎样退订"，其实是担心被偷偷扣钱。这里直接说清楚：**搬瓦工不会自动从你的信用卡或 PayPal 扣款**。官方明确说过他们不存储支付信息，技术上就没法自动扣。

不过有个例外——如果你账户里预充了余额，到期出账单时系统会直接扣余额续费。这个要注意。

续费账单的逻辑是这样的：VPS 到期前 7 天，系统自动生成一张续费账单，同时发邮件通知你。如果账户有余额，自动扣掉；如果没余额，邮件里有付款链接，你决定要不要付。7天内没付款，VPS 先暂停；再拖一段时间账单被取消，VPS 和所有数据彻底删除。

所以不想续费的话，最简单的方法就是：**到期后别付款就行**。你还有7天缓冲期，反悔了还能补交。

## 第一种情况：只是不想续费了，怎么取消

如果你只是不想续了、让 VPS 自然到期消失，操作很简单。

**步骤如下：**

1. 登录搬瓦工后台（官网镜像地址 bwh81.net）
2. 点顶部菜单 **Services → My Services**
3. 找到你的 VPS，点 **Manage**（或者直接找 cancel 按钮）
4. 页面里找到 **Request Cancellation** 按钮，点击
5. 弹出确认页面，点 **Submit request** 确认

这时候续费状态会从绿色的 ON 变成红色的 OFF。

注意一点：取消续费和退款是两码事。取消续费只是告诉系统"别再给我发账单了"，现有服务一直用到到期，不退钱。

顺便说一下，如果手滑关错了想恢复，在同样的页面点 **Restore** 就能重开续费通知，但必须在 VPS 到期**之前**操作——一旦到期，账号下的数据就没了，想恢复也来不及了。

## 第二种情况：想退款，这 8 个条件缺一不可

搬瓦工有30天退款政策。但很多人搞错一个关键细节：**这 30 天是账户注册时间，不是购买 VPS 的时间**。

举个例子——你三个月前注册了账户，昨天才买了第一台 VPS，直接不符合条件。

全部退款条件列出来：

| 条件 | 说明 |
|------|------|
| 账户注册时间 ≤ 30 天 | 最重要的一条，超过就彻底没得退 |
| 账户信誉良好 | 没欠款、没违反服务条款 |
| 没有发起过支付争议 | 没通过 PayPal 或信用卡公司发起过 dispute |
| IP 地址干净 | VPS 分配的 IP 不在黑名单，没有因封禁免费更换过 IP |
| 当月流量使用 < 10% | 套餐是 1TB，用了不能超 100GB |
| 账户下服务数量 ≤ 3 个 | 包括已取消的服务也算进去 |
| 总付款金额 ≤ $100，次数 ≤ 10 次 | 实际上稍超也有成功退款的，但官方条款如此 |
| 没有退款记录 | 同一账户或关联账户只能用一次退款权利 |

不满足以上任何一条，系统直接拒，没有商量余地。

## 申请退款的具体操作步骤

满足条件的话，操作如下：

1. **先备份数据**。退款一旦提交，账户下所有 VPS、快照、数据立刻删除，永久不可恢复。这一步别省。
2. 登录搬瓦工后台
3. 访问退款申请页面（登录后在账户设置里找，或直接访问 `bwh81.net/clientarea.php?action=refund`）
4. 页面上如果满足条件，会显示退款按钮；点击提交申请
5. 系统自动生成一个工单，工单里**只回复 `I agree`**——就这两个词，不要加引号、标点或任何多余内容，系统是自动识别的
6. 等几分钟到几小时，退款到账

退款到账方式：PayPal 付款的退回 PayPal，信用卡付款的退回信用卡（可能需要 3-14 天到账）。支付宝付款的情况稍微特殊——钱会退到搬瓦工账户余额，**不能提现，只能用来买新服务或续费**。

退款处理后，这个账户还可以继续买搬瓦工的产品，但之后就不再有退款资格了。

## 退款条件不满足怎么办

不满足退款条件的常见情况，处理思路不一样：

**流量用超了 10%**：这条没有弹性，系统自动检测。唯一办法是找客服发工单解释，但成功率很低，基本别指望。

**账户超过 30 天**：从官方渠道完全没法退款。可以发工单试一下，但能退的可能性极小。

**忘记备份数据了**：……这个就没办法了，所以一定要先备份再提交。

**想退款但只退其中一台 VPS**：不支持。搬瓦工退款是整个账户一起退，所有服务全部删除。如果只是不想要某台，直接取消续费就行，等它到期。

## 常见问题

**Q：搬瓦工取消续费会马上停服吗？**
不会。取消续费只是关掉续费通知，当前已付费的服务照常使用到原本的到期日期。

**Q：我账户里充了余额，取消续费后余额怎么办？**
余额会留在账户里，可以用来买其他服务，但不能提现。如果打算彻底不用了，可以在退款前把情况说明发工单，但余额一般来说是不退的。

**Q：退款之后同一个账户能重新买吗？**
可以。退款后账户还在，可以继续下单购买新套餐，只是这个账户不再有退款资格。

**Q：不满足退款条件，能不能找客服特殊处理？**
可以试试发工单解释情况，但成功概率不高。满足官方退款条件的走自助流程最快；不满足的，客服也通常无法破例。

**Q：搬瓦工关闭了续费通知，到期后数据还在吗？**
关掉续费通知后，到期出账单、账单7天内没付款，服务暂停，数据还在但无法使用；再过一段时间账单被取消，VPS 和数据彻底删除。所以如果只是想停掉但保留数据更长一点时间，还是保持续费通知开启、到期后7天内不付款更稳妥。
